we will build a surrogate model of the Rosenbrock function
and a constraint related to an Rosenbrock-based optimization problem.

    expressions={"z": "(1-x)**2+100*(y-x**2)**2", "c": "((x-1)**2+(y-1)**2)**0.5"},
